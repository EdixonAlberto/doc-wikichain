## Contribuir

- Crear ruta del nuevo documento en [./src/vuepress/config.js](./src/.vuepress/config.js)

- Crear nuevo documento markdown

- Formatear los archivos creados ejecutando `yarn lint`

- Crear pull request hacia el branch **dev**
